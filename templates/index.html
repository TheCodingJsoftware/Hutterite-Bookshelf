Read Me

Welcome to the Hutterite Bookshelf App! Here you can join, host and share sing-along sessions with friends, family and communities in addition to being able to install the app for offline use. Follow the instructions below to get started:

    Update Regularly
    • Periodically check for updates to ensure you have the latest version of the Hutterite Bookshelf app to take advantage of the latest features and improvements.
    How do I install for offline use?
    • Please see the instructions here
    What is a 'sing-along?'
    • A sing-along is a session where participants can join and synchronize to sing songs together, led by a host who controls the song selection. You can find more information about using sing-alongs here.

How to Install Hutterite Bookshelf

The Hutterite Bookshelf can be installed on various devices and browsers for offline use. Follow the instructions below for your specific device and browser:

Android
    Google Chrome
    • Open the app in Google Chrome and navigate to the Hutterite Bookshelf app URL.
    • Tap the three-dot menu icon in the top-right corner.
    • Select "Add to Home screen".
    • Confirm by tapping "Add" in the prompt that appears.
    • The Hutterite Bookshelf app will now be added to your home screen and can be used offline.
    Firefox
    • Open the app in Firefox and navigate to the Hutterite Bookshelf app URL.
    • Tap the three-dot menu icon in the bottom-right corner.
    • Select "Install" or "Add to Home screen".
    • Follow the prompts to add the app to your home screen.
    • The Hutterite Bookshelf app will now be added to your home screen and can be used offline.
iOS
    Safari
    • Open the app in Safari and navigate to the Hutterite Bookshelf app URL.
    • Tap the Share button (square with an uHutterite Bookshelf apprd arrow) at the bottom of the screen.
    • Scroll down and select "Add to Home Screen".
    • Confirm by tapping "Add" in the top-right corner of the prompt that appears.
    • The app icon will be added to your home screen, and you can launch it from there and use it offline.
Desktop
    Google Chrome
    • Open the app in Google Chrome and navigate to the Hutterite Bookshelf app URL.
    • Click the Install icon (plus in a circle) in the address bar or the three-dot menu icon in the top-right corner.
    • Select "Install Sing Along".
    • Confirm by clicking "Install" in the prompt that appears.
    • The Hutterite Bookshelf app will be installed as a standalone app and can be launched from your desktop or start menu.
    Microsoft Edge
    • Open the app in Microsoft Edge and navigate to the Hutterite Bookshelf app URL.
    • Click the Install icon (plus in a circle) in the address bar or the three-dot menu icon in the top-right corner.
    • Select "Install Sing Along".
    • Confirm by clicking "Install" in the prompt that appears.
    • The Hutterite Bookshelf app will be installed as a standalone app and can be launched from your desktop or start menu.
    Firefox
    • Open the app in Firefox and navigate to the Hutterite Bookshelf app URL.
    • Firefox does not fully support desktop Hutterite Bookshelf app installation. However, you can create a shortcut:
    • Click the three-dot menu icon in the top-right corner.
    • Select "Save Page As" and choose the location.
    • Manually create a shortcut on your desktop.
    • You can use the shortcut to access the app quickly, but it won't be a standalone Hutterite Bookshelf app like in Chrome or Edge.

Sing-Along Sessions

  • To view the join/create sing-along options press the icon in the top left corner of the home page.
    • Start Sing-Along:
    • An internet connection is required to use sing-along.
    • You have the option to plan songs before starting a session. You can add songs from the home page using the checkbox on the right of the song. The search functionality is highly efficient.
    • Once you have selected songs, click "Create Sing Along". Enter a code name that you can share with others. The password is for the host only, in case you get logged out. This password is used in the "Join Sing Along" section where you need to manually enter the information and press "Join".
    • You can choose to make the session private or public. If it is public, anyone can join your sing-along. If it is private, only the host can share the session.
    • To end the sing-along session, click the button on the top left inside the song page and select "End Sing Along".
    Join Sing-Along:
    • To join a sing-along, simply open the link you have been provided, select from the list of public sing-alongs, or manually enter the code you have been given.
    • Once you have joined, you cannot select any songs. You will be synced to what the host is currently viewing, and only the host can change songs.
    • To leave the sing-along, click the button on the top left inside the song page and select "Leave Sing Along".
